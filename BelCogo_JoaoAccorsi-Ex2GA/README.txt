Entrega 2 GA - Bel Cogo e João Accorsi

Nesta pasta estão a definição da gramática, arquivo de texto para teste da gramática e também um .png com uma foto da árvore gerada pela gramática.
Segue abaixo o nome dos respectivos arquivos:
  - SytacticAnalyser.g4
  - sample.txt
  - antlr4_parse_tree.png

Para fazer outros testes com essa gramática, basta alterar o arquivo sample.txt e rodar o comando antlr4-parse SyntacticAnalyser.g4 prog -gui sample.txt dentro dessa pasta.

Link para o github com o analisador lexico e o analisador semântico -> https://github.com/belcogo/compilers
